Empresa transportadora de Colombia

== Description ==

Integración de la transportadora envi@ como método de envío para Woocommerce. Actualmente esta versión solo permite calcular el costo de envío.

== Installation ==

1. Descargar el plugin
2. Ingrese a su administrador de wordpress
3. Ingrese al menu plugins / añadir nuevo
4. Busque por el nombre **Shipping Envi@ Colvanes Woocommerce**
5. Instale y active el plugin y siga el proceso de configuración.
6. Establezca los detalles del envío del producto, ver captura de pantalla.
